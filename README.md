# 🚴 Bike Sales Data Analysis Project (Excel Dashboard)

## 📌 Project Overview
This project analyzes bike sales data to uncover customer purchasing patterns, demographic trends, and revenue insights. The objective was to transform raw sales data into meaningful business insights using Microsoft Excel.

The final output is an interactive dashboard that enables quick decision-making through visual analytics.

---

## 🎯 Business Problem
The company wanted to understand:

- Which customer demographics purchase bikes the most
- How income level influences buying decisions
- The impact of commute distance on bike purchases
- Key sales trends across different customer segments

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning Techniques
- Data Visualization

---

## 🔄 Data Preparation Process
The dataset was cleaned and prepared using the following steps:

- Removed duplicates
- Standardized categorical values (e.g., marital status, gender)
- Created calculated columns where necessary
- Ensured consistent formatting for analysis
- Structured data for pivot table reporting

---

## 📊 Dashboard Features
The interactive dashboard includes:

- Sales breakdown by income level
- Purchase trends by age group
- Analysis by marital status
- Commute distance impact on bike purchases
- Dynamic filtering using slicers

The dashboard allows users to interact with the data and quickly identify trends.

---

## 📈 Key Insights
Some important findings from the analysis:

- Middle-aged customers showed the highest purchase rate
- Higher income customers were more likely to purchase bikes
- Customers with shorter commute distances had increased purchase frequency
- Marital status influenced purchasing behavior
